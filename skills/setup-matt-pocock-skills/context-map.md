# Context Map

Purpose: route the agent to the right context document.

## Default route

- Start with `.github/context/shared-context.md`.

## Optional module routes

Add module context docs only when a module has distinct concepts or language.

Examples:

- `.github/context/design-system-context.md`
- `.github/context/app-shell-context.md`
- `.github/context/forms-context.md`

## Notes

- This file is navigation, not a glossary.
- Keep it short and link-oriented.
